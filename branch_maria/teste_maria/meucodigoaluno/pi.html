Projeto BrainCode

Plataforma virtual de aprendizagem com funcionalidades desenvolvidas em PHP para cadastro, edição, exclusão e consulta de informações. Alunos têm acesso a materiais de estudo e ferramentas para tarefas e notas; professores podem criar atividades, atribuir notas e gerenciar conteúdo. Ambos os perfis requerem autenticação para segurança. O administrador possui controle total, incluindo gerenciamento de usuários e conteúdos.
